# 🎮 2D Horror Puzzle Game

## 📌 Overview

A narrative-driven psychological horror game developed in Unity, designed to immerse players through atmosphere, environmental storytelling, and tension-building gameplay. The project focuses on creating fear through uncertainty, subtle cues, and controlled pacing rather than relying on traditional jump scares.

The game features puzzle-based progression, where players explore interconnected environments, uncover narrative elements, and interact with objects to advance the story. Core systems were built using event-driven architecture, enabling dynamic in-game events and responsive player interactions.

Special attention was given to lighting, sound design, and scene composition to enhance immersion and emotional impact. The project also emphasizes clean code structure and performance optimization to ensure smooth gameplay and scalability for future expansion.

## 🚀 Features

- Puzzle-based progression system
- Event-driven gameplay triggers
- Atmospheric sound design
- Custom 2D assets and environments

## 🛠️ Tech Used

* Unity (2D)
* C#
* Unity Physics & UI System

## 🧠 What I Learned

+ Developed an understanding of psychological horror design, focusing on building player tension through environmental storytelling, lighting, and controlled pacing rather than relying on jump scares.
+ Learned how to design player-driven experiences, where uncertainty and limited information create a sense of fear and curiosity
+ Implemented event-driven systems in Unity, including trigger-based interactions, scripted sequences, and state-based progression logic
+ Gained experience in structuring modular and reusable C# scripts for better scalability and maintainability
+ Applied game physics and collision systems to create responsive and believable interactions within the game world
+ Worked on performance optimization techniques, including reducing unnecessary updates, optimizing scripts, and managing assets efficiently
+ Improved skills in debugging and problem-solving, especially when handling complex gameplay interactions and unexpected behavior
+ Learned the importance of user experience and feedback loops, ensuring gameplay feels smooth, intuitive, and engaging
+ Developed a stronger grasp of Unity’s core systems such as scene management, UI integration, and audio design
+ Practiced balancing technical implementation with creative design, ensuring mechanics support the overall atmosphere and narrative
